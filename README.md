# 청각장애아동을 위한 한글 학습프로그램 『울림』

아동들은 동요를 들으며 한글을 자연스럽게 습득합니다.   
하지만 청각장애아동은 동요를 듣기 어려워 한글능력이 또래에 비해 떨어집니다.   
촉각과 시각으로 동요를 느끼게 해주자는 생각을 했습니다.
## Installation

HTML/CSS/JS 하드코딩 프로젝트
